Software


  R package:

  • amgfl (Github)
    • Trend estimations by semiparametric additive model with graph trend filtering via generalized fused Lasso.
    • Building simultaneous confidence intervals via the bootstrap.
  • GFLglm (Github)
    • Coordinate descent algorithm to solve generalized fused Lasso problem for generalized linear models.
    • This package includes the R package GFLlogit.
  • GFLlogit (Github)
    • A multivariate trend filtering for binary data via generalzied fused Lasso.
  • GGFL (Github)
    • An estimation for discrete varying coefficient model via generalized group fused Lasso.
    • A grouping for linear regression models based on adjacent relationship among groups.
  • GRRMSC (Github)
    • A fast optimization of ridge parameters for generalized ridge regression via model selection criterion minimization methods.
    • This package covers optimizations for a partial generalized ridge regression and an additive model.
  • trec (Github)
    • Trend estimation and classification to support integrated assessment of the marine ecosystem and environmental factors.
